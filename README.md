# パーソナル・タロット占い

Claude Code で作ったブラウザで動くタロット占いアプリ。
名前・生年月日・お悩みを入力 → 22枚の大アルカナから直感で3枚選ぶ → 過去・現在・未来の解釈とライフパスナンバーが表示される。

## デモ

→ https://ryoya9595.github.io/tarot-app/

## 機能

- 大アルカナ22枚から3枚を選ぶ「直感選択」UI
- 占うテーマ選択（恋愛運 / 仕事運 / 総合運）
- 入力した「お悩み」を結果に反映
- 生年月日からライフパスナンバーを自動算出（1〜9, 11, 22, 33対応）
- カードめくりアニメーション、神秘的な紫×ゴールドのデザイン
- スマホ対応（レスポンシブ）

## 技術

- 単一HTMLファイル（外部ライブラリ・CDNなし）
- バニラJavaScript
- CSS のみで星空背景・カードフリップアニメーション

## ローカルで動かす

```bash
# index.html をブラウザで開くだけ
open index.html
```
